安装与配置 Hexo

概述

Hexo 是一个轻量的静态博客框架。通过Hexo可以快速生成一个静态博客框架,仅需要几条命令就可以完成,相当方便。

配置环境

系统环境： OS X / Linux / 类 UNIX 系统（我没使用 Windows 系统，所以不作相关说明）

安装 Node

OS X

  • 直接下载 pkg 安装包安装
  • 使用 Homebrew
  • 使用 MacPorts

使用软件包管理器

  • Debian 系
  • Redhat 系

源码编译

  • 下载源码归档，并切换到所在目录

其他发行版请自行 Google

安装 Git

  • OS X ：使用 Homebrew, MacPorts 或安装 Xcode 获得。
  • Debian 系
  • Redhat 系
